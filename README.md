# RoPE Visualized

Static, dependency-free interactive demo for teaching rotary positional embeddings.

## Open locally

Open `index.html` in a browser.

## Publish

Upload this folder to any static host, such as GitHub Pages, Netlify, Vercel static output, or an LMS file area.

## Notes

- The token vectors are deterministic demo vectors, not extracted model weights.
- The absolute sinusoidal position addition and RoPE pair rotation use real formulas.
- The comparison heatmaps show cosine similarity over the transformed output vectors.
- Keyboard shortcuts: left/right arrows move through the story.
